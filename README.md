# Kaggle_Disaster-Tweets_NLP
# Kaggle Disaster Tweets NLP  An NLP model predicting if a tweet is about a real disaster.   - Data: Text cleaned from URLs &amp; punctuation - Vectorizer: TF-IDF - Classifier: Logistic Regression - Score: 0.78 Accuracy  A clean and efficient baseline machine learning project built using Python, Pandas, and Scikit-Learn.
